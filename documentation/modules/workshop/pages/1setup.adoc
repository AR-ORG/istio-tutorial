== Prerequisite CLI tools

The following CLI tools are required for running the exercises in this tutorial. Please have them installed and configured before you get started with any of the tutorial chapters.

[cols="4*^,4*.",options="header"]
|===
|Tool|macOS|Fedora|Windows

|Kubernetes
| `brew install kubernetes-cli`
| https://github.com/kubernetes/kubernetes/releases[Kubernetes Releases]
| https://github.com/kubernetes/kubernetes/releases[Kubernetes Releases]

| oc (Openshift client)
| https://github.com/openshift/origin/releases[Openshift Releases]
| `brew install openshift-cli`
| https://github.com/openshift/origin/releases[Openshift Releases]

| `git`
| https://git-scm.com/download/mac
| `dnf install git`
| https://git-scm.com/download/windows

| `Docker`
| https://docs.docker.com/docker-for-mac/install
| `dnf install docker`
| https://docs.docker.com/docker-for-windows/install/

| `stern`
| `brew install stern`
| https://github.com/wercker/stern[stern]
| https://github.com/wercker/stern[stern]


| httpie
| `brew install httpie`
| `dnf install httpie`
| https://httpie.org/[httpie]

| https://github.com/JoeDog/siege[siege]
| `brew install siege`
| `dnf install siege`
| https://github.com/JoeDog/siege[siege]

| kubectx and kubens
| `brew install kubectx`
| https://github.com/ahmetb/kubectx
